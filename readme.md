# erigere

D3.js を使ったシンプルなマインドマップツールです。  
ブラウザ上でノードの追加・編集・削除を行い、データは `localStorage` に保存されます。

## ファイル構成

- `index.html`: エントリーポイント
- `script.js`: マインドマップの描画・操作ロジック
- `style.css`: 画面スタイル

## 使い方

1. `index.html` をブラウザで開く
2. 中央ノード（中心トピック）から操作を開始する

## 操作方法

- **ノード選択**: ノードをクリック
- **子ノード追加**: ノードをダブルクリック
- **ノード編集**: ノード選択後 `Enter`、または追加直後に自動編集
- **ノード削除**: ノード選択後 `Delete` / `Backspace`  
  （中心トピックは削除不可）
- **ノード移動**: ドラッグ

## データ保存

- 保存先: ブラウザの `localStorage`（キー: `mindmap-data`）
- 読み込み: ページ再読込時に自動復元
